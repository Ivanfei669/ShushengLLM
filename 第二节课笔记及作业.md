### 2. 部署InternLM2-Chat-1.8B模型进行智能对话
#### 2.1 配置基础环境
- 在Intern Studio配置开发机。
- 使用Cuda11.7-conda镜像，选择10% A100 * 1资源配置。
- 进入开发机后，在终端配置环境。

```bash
studio-conda -o internlm-base -t demo
# 或者
conda create -n demo python==3.10 -y
conda activate demo
conda install 相关依赖
```

- 激活新的conda环境，安装必要的包。

#### 2.2 下载InternLM2-Chat-1.8B模型
- 创建文件夹，进入目录。
- 下载模型参数。

#### 2.3 运行cli_demo
- 执行demo程序，与模型互动。
